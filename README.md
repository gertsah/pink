# ПИНК · Премиум e-commerce

Сайт ПИНК — премиальная женская одежда, сумки, обувь и бижутерия. Коллекция «Весна-Лето 2026».

## Структура

- `index.html` — главная (hero-стек с sticky reveal, новинки, луки, категории, истории, бутик)
- `pages/clothing.html` — одежда
- `pages/bags.html` — сумки (с hover-swap product → on-model)
- `pages/shoes.html` — обувь (с hover-swap)
- `pages/jewelry.html` — бижутерия
- `pages/looks.html` — луки с hotspots
- `pages/photography.html` — полноэкранная фотогалерея
- `pages/films.html` — видео-кампании
- `pages/product.html` — карточка товара (PDP)
- `styles/main.css` — все стили
- `scripts/main.js` — интерактив

## Локальный запуск

```bash
npx serve -l 8000
```

Открыть `http://localhost:8000/`.

## Деплой

Сайт развёрнут через GitHub Pages.
